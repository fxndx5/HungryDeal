# Frontend - HungryDeal

Capa de presentación del proyecto, dividida en tres partes:

## Estructura

```
frontend/
├── web/       # Aplicación web con Next.js 14 (App Router)
├── mobile/    # Aplicación móvil con React Native + Expo
└── shared/    # Código compartido entre web y móvil
    ├── api/   # Clientes HTTP para comunicarse con el backend
    └── types/ # Interfaces y tipos TypeScript comunes
```

Cada subcarpeta tiene su propio README con más detalle.
